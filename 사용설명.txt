Hi Ultimate 단타 탐색기 V7

[실행]
1. 압축을 풉니다.
2. PC에서는 index.html을 더블클릭합니다.
3. 휴대폰에서는 GitHub Pages 저장소에 파일 5개를 모두 올립니다.
   - index.html
   - manifest.json
   - sw.js
   - icon-192.svg
   - icon-512.svg

[주요 기능]
- 한국·미국 주식 통합 TOP 20
- 장전 후보 TOP 5 및 브리핑
- 뉴스·거래량·이동평균선·수급·미국 증시·거래대금 점수
- 거래량 급증 및 골든크로스 표시
- 관심종목 저장과 JSON 내보내기
- 브라우저 알림 시험
- 점수 배점 변경
- 실시간 API 주소 저장
- PWA 설치 및 오프라인 캐시

[중요]
현재 종목 데이터는 화면과 기능 시험용 데모 데이터입니다.
실제 매수 추천이 아니며 수익을 보장하지 않습니다.
실시간 데이터는 증권사 API, 뉴스 API, 수급 데이터 서버 연결이 필요합니다.
