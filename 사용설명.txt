Hi Ultimate 단타 탐색기 V8 실전 연결판

업데이트 방법
1. 기존 pro-v7 저장소 파일을 모두 삭제할 필요는 없습니다.
2. 이 ZIP의 index.html, manifest.json, sw.js, icon-192.svg, icon-512.svg를 업로드합니다.
3. 같은 이름의 기존 파일은 Replace files로 덮어씁니다.
4. GitHub Pages 배포 후 Ctrl+F5로 새로고침합니다.

실시간 연결
- 설정 → 데이터 방식 → Alpha Vantage 선택
- 발급받은 API 키 입력 → 연결 설정 저장
- 연결 시험
- 후보 화면에서 실시간 갱신

중요
- API 키는 브라우저 localStorage에만 저장하며 GitHub 파일에 포함하지 않습니다.
- 한국투자증권 API는 접근 토큰과 비밀키 보호를 위해 별도 서버가 필요합니다.
- 자동 주문 기능은 포함하지 않습니다.
- 후보 점수와 분석은 참고용이며 수익을 보장하지 않습니다.
